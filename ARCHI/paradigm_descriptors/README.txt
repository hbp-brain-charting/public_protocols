The user shall run firstly the script "parser_eprime_logtxt_ARCHI.py" in order to extract the values concerned to the relevant paradigm descriptors.
Afterwards, the user shall run the script "paradigm_descriptor_ARCHI.py" in order to generate the descriptors as regressors-of-interest to be used for the GLM estimation.
